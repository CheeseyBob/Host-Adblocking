--- Host-Adblocking ---
Blocking ads on a system level without needing an ad-block browser extension.


This is a convenience script for using StevenBlack's unified hosts blacklist:
https://github.com/StevenBlack/hosts

Setup:
* Put your own base hosts in a file named 'my-hosts' in the script directory.
* Put your own hosts blacklist in a file named 'my-blacklist' in the script directory.

Use:
* Run the script 'host-adblocking'.

You can use the script to make a backup of your /etc/hosts file.
You can use the script to check before changing your /etc/hosts file.
